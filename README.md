# Генератор PDF протоколів для Пліч-о-Пліч

Генератор PDF протоколів зі спортивного орієнтування (IOF XML 3.0) для національного проекту Пліч-о-Пліч

CLI-інструмент для перетворення результатів з формату IOF XML 3.0 (експорт з MeOS) у PDF-протоколи:

- Індивідуальний протокол (по класах)
- Командний протокол (окремо чоловіки та жінки)

## Вимоги

- Node.js 18+
- npm

## Встановлення

```bash
npm install
